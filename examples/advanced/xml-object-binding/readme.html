XML object binding examples

Overview

The XML object binding examples demonstrate how to use open source libraries to achieve the following:
  1. Convert Java beans to XML.
  2. Send the XML to the Mule ESB.
  3. Receive XML resposne from the Mule ESB.
  4. Convert XML response to Java beans.

These examples use a server with services that use the mule-cics transport. The client sends XML to a mule-cics service and receives XML as response. The request and response XML have to conform to the schema, which are publicized in the WSDL or XSD file.

Running the examples

  1. Start the Mule server
    ${mule-cics}/examples/xml-object-binding/mule-server>run
  2. Run any of the clients
    ${mule-cics}/examples/xml-object-binding/client-castor>mvn test
The client will convert Java bean to XML and send the XML to the Mule server. The server will send back a dummy XML response, and client will convert it to Java bean.

Note: The examples require to use Maven (http://maven.apache.org)

Common Steps to develop the client

The common development steps to be followed to develop a client using open-source libraries such as XMLBeeans, Castor, JIBX, JAXP are explained below:
  • Place XSD files in the Maven project under src/test/resources.
  • Add plugin in pom.xml to generate Java beans from XSD schema.
  • Generate java beans from the XSDs using maven.
    #mvn compile
  • Write the code to :
    • Create an instance of generated Java bean and copy data in it.
    • Use the open source library API to convert Java bean to XML and send it to server.
    • Use the open source library API to convert response XML back to request object.

Please see the following folders for the actual source code.

  • examples/xml-object-binding/xmlbeans : Example using XMLBeans.
  • examples/xml-object-binding/castor: Example using Castor
  • examples/xml-object-binding/jibx folder: Example using JIBX
  • examples/xml-object-binding/jaxp folder: Example using JAXB
Note: The Castor example is not working. (The Castor client is unable to convert a XML value of type xsd:long)
