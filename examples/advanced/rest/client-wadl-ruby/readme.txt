README

This sample requires the rest-open-uri- package.

To install this package, first set the http proxy/password
and then give the gem install command as below:


set http_proxy=http://proxy.example.com:80
set http_proxy_user=username
set http_proxy_pass=password 

gem install rest-open-uri-1.0.0.gem

Run client:
    #ruby sampleCommand.rb
